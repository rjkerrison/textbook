# Graphics

Graphics on the web can go from the classic to the highly advanced, and include the following:

- GIFs
- JPEGs
- PNGs
- Flash
- Java Applets
- MPEGs
- SVGs
- Canvas 2d rendering
- WebGL
- WebGL2

Some of these are deprecated; for instance, Flash support has been removed from modern browsers.
